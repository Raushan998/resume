# Raushan Kumar Raman — Interactive Resume

A retro Mario-style side-scrolling platformer that doubles as my resume.
Run right, collect skill coins, hit `?` blocks to read about each job, reach the flag to see contact info.

**Live site:** https://raushan998.github.io/resume/

## Controls
- `← →` or `A / D` — move
- `Space` / `↑` / `W` — jump
- On mobile: virtual D-pad + jump button

## Tech
Single self-contained `index.html` — no build step, no dependencies (just Google Fonts via CDN).
Pure HTML / CSS / Canvas 2D / vanilla JavaScript.
